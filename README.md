# ⚽ FootballTennis

## 🇬🇧 English

FootballTennis is a web application for managing football tennis (nohejbal) tournaments.  
The system allows users to create tournaments, manage teams and players, record match results and automatically calculate standings.  
The application follows real tournament rules including sets, points and ranking logic.  
The project was created primarily as a personal learning project and portfolio demonstration.  
Its goal is to practice modern web development with ASP.NET and clean application architecture.

---

## 🇨🇿 Česká verze

FootballTennis je webová aplikace pro správu nohejbalových turnajů.  
Umožňuje vytvářet turnaje, spravovat týmy a hráče, zapisovat výsledky zápasů a automaticky počítat tabulku.  
Aplikace respektuje reálná pravidla turnaje včetně setů, bodů a výpočtu pořadí.  
Projekt vznikl jako osobní studijní a portfoliový projekt.  
Cílem bylo procvičit vývoj webových aplikací a návrh čisté architektury.
